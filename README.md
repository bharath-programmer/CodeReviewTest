# Employee Management System

A small React + TypeScript + Vite application used as a clean baseline for demonstrating an AI Code Review Agent.

## Features

- Login page (UI only)
- Dashboard
- Employee list with search
- Employee details
- Add employee

Data is stored in mock JSON and persisted with `localStorage`. There is no backend or real authentication.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start the development server
- `npm run build` — type-check and build for production
- `npm run preview` — preview the production build
